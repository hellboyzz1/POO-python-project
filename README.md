#  Sistema de Gerenciamento de Alunos (POO em Python)

Projeto desenvolvido em Python utilizando Programação Orientada a Objetos (POO) para representar e gerenciar dados de um aluno.

## 🧠 Descrição

Este sistema permite cadastrar um aluno, adicionar notas, calcular a média e verificar automaticamente se o aluno foi aprovado ou reprovado com base no desempenho.

O projeto foi criado com foco na prática de conceitos fundamentais de programação.

## 🚀 Funcionalidades

- Criação de aluno com:
  - Nome
  - Idade
  - Matrícula
- Adição de notas
- Cálculo automático da média
- Verificação de aprovação (média ≥ 5)
- Exibição completa dos dados do aluno

## 🛠️ Tecnologias utilizadas

- Python 3
- Programação Orientada a Objetos (POO)

## 📚 Conceitos aplicados

- Classes e objetos
- Métodos
- Encapsulamento
- Listas
- Estruturas condicionais (`if/else`)
- Funções internas da linguagem (`sum`, `len`)
